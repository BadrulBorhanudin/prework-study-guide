Git


Generate an SSH Key


  1. Enter ssh-keygen -t ed25519 -C “your_email@example.com” in your terminal to create a new SSH key.
  2. When prompted, save your key in the default file location and create a secure passphrase.

Add the SSH Key to the ssh-agent


  1. Enter eval “$(ssh-agent -s)” in your terminal to start the ssh-agent.
  2. Add your private SSH key to the ssh-agent using the ssh-add command

Add the SSH Key to GitHub


  1. On GitHub, go to Settings/SSH and GPG keys
  2. Click New SSH key

On GitBash


  • cd prework-study-guide: make sure we're in the right folder
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it [for example: feature/starter-code]
  • git status: make sure we're in a new branch we've just created
  • git add -A: to save all files
  • git commit -m "short message and descriptive"
  • git pull origin main: to make sure we have all the latest changes and commit
  • git push origin feature/starter-code

On GitHub


  • Under repository, click on Compare & pull request
  • Give a descriptive title
  • Add a comment briefly explaining the changes we made
  • Create pull request to merge with the main branch
  • Ask another colleague to review our changes and then they approve the merge request
  • Select Merge pull request to merge feature/starter-code into the main branch and Confirm merge
